# Design Brief — 에디의 하루 (플래너 통합)
생성일: 2026-07-11

## 목적
ADHD 사용자가 **계획(캘린더)과 실행(에디 동행)**을 한 앱에서 이어가도록, 시간맹을 보정하고 실패해도 이탈하지 않게 만드는 모바일 앱 화면.

## 대상 사용자
- **P1** 지각·미루기형 직장인(28세, 진단), **P2** 계획 마비형 대학생/프리랜서(24세)
- 헬스 리터러시 보통, 기기 친숙도 높음, 결정 피로·거절민감성(RSD) 높음
- 사용 맥락: 아침 출근 준비 중(급함), 이동 중, 잠들기 전(피곤) — **한 손·짧은 주의력** 전제

## 화면 스펙
- 플랫폼: iOS / Android 모바일 앱 (기기 TFT 아님)
- 기준 해상도: **390×844pt (iPhone 14 기준)**, 반응형
- 터치: 예 — 터치 타깃 최소 44×44pt
- 위젯: 홈스크린 위젯(오늘 뷰·출발 카운트다운) 별도 스펙

## 우선 설계 화면 (3순위)
1. **오늘 탭** — 에디 시계 + 다음 출발 카운트다운 + 오늘 타임라인 + 할 일 3개 + 복약 카드
2. **할 일 → 타임블록 제안 모달** — 에디가 빈 시간에 블록 제안, 수락/수정/이월
3. **실패·회복 모달** — 놓침 시 자책 없는 회복 경로(지금/저녁/내일/패스)

## 표시 데이터
| 항목 | 형식 | 상태 표현 | 비고 |
|------|------|-----------|------|
| 에디 시계 | 시:분 | 실제보다 5~10분 빠름 | **정확 오프셋 비노출** |
| 출발 카운트다운 | 남은 시간 | 여유(안심)/임박(강조) | 이동+준비 역산 |
| 타임블록 | 시간 블록 | 예정/진행/완료/놓침/이월 | 색+아이콘 병행 |
| 할 일 | 텍스트 리스트 | 우선 3개만 | 과다 시 접기 |
| 복약 | 카드 | 예정/완료/놓침 | 놓침도 자책 없이 |
| 스트릭 | 회복형 게이지 | 리셋 없음 | '회복' 개념 |

## 브랜드 / 컬러
> 에디의 하루는 SuGenTech 의료 제품과 **별도 브랜드**. 임상 톤(SU Red 다크카드)이 아니라 **따뜻하고 안심되는 캐릭터 친화 팔레트** 권장. 아래는 제안값 — 멍이 조정 가능.

- **톤 앤 매너**: soft / warm / 안심(non-clinical). 죄책감 유발 톤 절대 금지
- **Primary (에디 시그니처)**: 따뜻한 코랄/앰버 계열 제안 (예: `#FF8A5B`) — 원본 시나리오의 노랑·주황 감정 톤 계승
- **Accent (완료·긍정)**: 부드러운 그린 `#5FBF8F`
- **Background**: 라이트 `#FBF7F2`(크림) / 다크모드 `#1C1B1F`
- **텍스트**: `#3A3A3A` (본문)
- **상태색**: 완료 그린 / 진행 앰버 / 놓침은 빨강 대신 **회색+아이콘**(질책 인상 회피)
- **폰트**: 라운드·친근한 산세리프(예: Nunito) + 숫자·카운트다운 강조용 모노(Space Mono)
- **캐릭터**: 에디 — 감정 표현이 풍부하되 과장 없는 마스코트. 상태(기쁨/담담/응원/굿나잇) 4종 최소

## 레퍼런스 / 키워드
- 무드: "옆에 있어주는 친구", body-doubling, 다정한 넛지
- 참고 방향: Finch(셀프케어)·Structured(타임블록)의 따뜻함 + Fabulous의 루틴 온보딩
- 피할 방향: 스트릭 강박형(Duolingo식 압박), 정보 과밀 대시보드

## 제약 조건
- **규제**: 비의료기기 — "진단·치료 아님, 복약은 리마인더" 면책 문구 포함
- **접근성**: WCAG 2.1 AA / 색+아이콘·텍스트 병행(색맹) / 폰트 200% / 애니메이션 감속 옵션
- **인지 부하 최소화**: 한 화면 정보 밀도 상한, 선택지 3개 이하 우선(NFR-A-002)
- **금지 패턴**: 자책 유발 문구, 빠른 시계 오프셋 노출, 자동 일정 배치(제안 후 수락만), 정서적 과의존 유도 카피
